## 📝 Заметки

- **БД**: новые таблицы создавать вручную через ALTER TABLE / CREATE TABLE — sync не справляется
- **Модели**: использовать `DataType.STRING` вместо `DataType.ENUM` — иначе 500 ошибки
- **git push**: пользователь пушит сам
