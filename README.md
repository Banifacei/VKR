# VKR
Дипломная работа на тему - Разработка отечественной веб-платформы интерактивного видеообучения 
Загрузка файлов поддерживает все популярные (и не очень типы файлов для видео).
Эти файлы будут воспроизводиться идеально в любом современном браузере (Chrome, Яндекс, Edge):
MP4 — самый популярный и надежный формат.
WebM — отличный формат для веба с высокой степенью сжатия.
OGG/OGV — свободный формат видео.
2. Профессиональные и мобильные
MOV — формат от Apple (iPhone/Mac). Обычно работает в Chrome без проблем.
MKV — современный «матрешка»-контейнер. Часто используется для качественных лекций и фильмов.
3. Старые форматы (Windows)
AVI — классика.
WMV — формат Windows Media.
